# ORM Learn - Spring Boot + Hibernate + MySQL

##  Project Description
This project demonstrates ORM (Object Relational Mapping) using:
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL Database

It performs a simple READ (SELECT) operation from a `country` table.

---

##  Technologies Used
- Java 17/21
- Spring Boot
- Spring Data JPA
- Hibernate ORM
- MySQL
- Maven
