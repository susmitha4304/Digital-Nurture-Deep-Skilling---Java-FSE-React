#  Spring Boot REST API - Hello World

##  Project Overview
This is a simple Spring Boot REST API project that demonstrates how to create and test a basic GET endpoint using Spring Web. The API returns a simple "Hello World!!" response.

---

##  Technologies Used
- Java
- Spring Boot
- Spring Web
- Maven
- IntelliJ IDEA
- Postman (for testing)
